R语言入门语法

基本操作

获取当前所在文件目录

getwd()

更改文件目录

setwd(dir = "C:Users/Desktop/Data")

  1. 注意“/”在编程语言中为路径地址的分割，而“\”为转义符号

  2. 当报错的时候要检查该路径下是否有该文件夹

查看当前文件夹下包含的文件

list.files()或者dir()

运算

赋值运算

R语言定义变量无需声明类型

x <- 3或者5 -> x或者x=3

  1. 局部变量
  2. 将3赋值给x，用小于号连接的减号赋值，（不推荐用等号赋值，可能与后面的假设检验冲突，也不推荐5 -> x这种赋值，因为降低了代码的可读性）

x <<- 5

这个是强制赋值（全局变量）

求和运算

y <- sum(1, 2, 3, 4, 5)

将1， 2， 3， 4 ，5的和赋值给y

求均值运算

z1 <- mean(c(1, 2, 3 ,4 ,5))
z1 <- mean(1:5)

将1， 2， 3， 4 ，5的均值赋值给z1

z2 <- mean(1, 2, 3, 4 ,5)

将1，2，3，4，5，的第一个数复制给z2

变量查看

ls()

查看当前工作区的变量

ls.str()

查看变量的详细信息该函数是ls()和str()的组合str()可以列出一个变量的具体信息

ls(all.names = TRUE)

列出以点开头的文件

清屏

Ctrl+L

清楚变量

rm(x)

清除工作区中的x变量

rm(x,y)

清除工作区中的x，y变量

rm(list = ls())

清除工作区中的所有的变量

寻找之前的命令

方向键控制

history(n)

列出历史记录 n为列出多少条，如不填则列出所有

保存工作空间

保存为.RData文件

save.image()

只会保存数据和绘图函数，图片会单独保存

退出

q()

包的安装（在线安装）

install.packages("")

引号内加入要安装的包

.libPaths()

显示库所在的位置

library()

显示库里有哪些包

包的使用

包的载入

library()
或者
require()

  1. 括号内填入要载入的包
  2. 区别是当载入一个未下载的包是require()会发出警告但是会继续执行程序
  3. 而library()会终止程序的运行

常用技巧

if(!require("cluster")) install.packages("cluster")

包的帮助

help(package="vcd")
library(help="vcd")

包的帮助

ls("package:vcd")

包中所有的函数

data(package="vcd")

列出包中包含的数据集

detach("package:vcd")

将包从内存中移除

remove.packages("vcd")

将包从硬盘中移除

包的移植

获取帮助

help.start()

打开帮助文档

help(sum)
或者
?sum

  1. 打开sum函数的帮助文档
  2. 如果搜索不都相关的文档，先需要载入包

args(sum)

快速了解sum的参数，而不需要打开帮助界面

example(mean)

给出mean函数的示例

vignette()

文档更细节

??qplot

输入两个问号不需要加载包就可以打开函数的帮助文档

help.search("heatmap")
或者
??heatmap

不知道函数的名字，用本地模糊搜索

apropos("sum")

列出所以包含sum关键字的内容

`apropos(“sum”,mod=“function”)

调整参数查询sum关键字的函数

RSiteSearch("matlab")
